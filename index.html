MENGHITUNG LUAS SEGITIGA

Berdasarkan gambar Diatas
maka Rumus Untuk Menghiting Luas dari sebuah Segitiga adalah :
1/2 dikali Alas dikali Tinggi (0.5 X A X T)










MENGHITUNG KELILING SEGITIGA

Berdasarkan Gambar Diatas (Tambah Bukan Kali)
Maka rumus untuk menghitung Kelililing Segitiga adalah :
sisi A ditambah sisi B ditambah sisi C (A + B + C)
